🐧 Penguin Computer

自分たちの自由なコンピュータ（Your own computer）

バージョンアップのたびにパソコンを買い替えるのは、もうやめよう。

Windows 10 のサポートは2025年10月に終了しました。Windows 11 は古いPCにインストールできません。でも、まだ使えるパソコンを捨てる必要はありません。

Linux という選択肢があります。

ペンギンコンピュータがやっていること

整備済み中古PCに Linux をインストールしてお届けしています。

  • 1台ずつ分解して内部クリーニング
  • SSD換装（起動・動作が10倍以上速くなります）
  • Linux インストール + チューニング
  • 日本語環境・必要なアプリをセットアップ済み

届いたらすぐ使える状態に仕上げています。
